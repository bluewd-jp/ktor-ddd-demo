# ktor-ddd-demo

Kotlin, Ktorに慣れることが主目的。

本レポジトリは [DddEndow/sample-ddd-laravel](https://github.com/DddEndow/sample-ddd-laravel) を参考にしたもの。

プロジェクト内容に関しては参考元を確認のこと。
